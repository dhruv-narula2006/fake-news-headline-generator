What to build?
A news headline generator that creates fake and funny news headlines using list of words.


How will it work?
1. We will create some lists of some words which will focus on:
      - Subject-- Who   and     What
      - Action-- What is the subject doing?
      - Places-- Where did it happen?
      - Object-- What is the subject doing to?

2. From the above lists created, we will randomly select one word from each list and combine them to create a news headline.

What to be used to select random words?
random module in Python can be used to select random words from the lists. We can use the random.choice() function to select a random word from each list and then combine them to create a news headline.

3. Ask the user if he wants to create another fake news or not. If yes, repeat the process, if no, exit the program.


--------------------------------
Concepts Used:
1. Lists: We will use lists to store the words for subjects, actions, places, and objects.

2. random.choice() function: We will use this function to select random words from the lists.

3. print() function: We will use this function to display the generated news headlines.

4. input() function: We will use this function to ask the user if they want to create another fake news headline or not.

5. Loops: We will use a loop to repeat the process of generating news headlines until the user decides to stop.

6. String concatenation or f-strings: We will use string concatenation or f-strings to combine the randomly selected words into a coherent news headline.


-------------------------------------------------------
Steps of Pseudocode:
1. Import the random module.

2. Create lists for subjects, actions, places, and objects- all with indian context.

3.Use random.choice() to select a random word from each list.

4. Combine the words int a headline using string formatting.

5. Print the generated headline.

6. Ask the user if they want to create another headline. If yes, repeat the process, if no, exit the program.
